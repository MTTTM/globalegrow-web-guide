# 兼容性

--- 

处理兼容性问题时，不要去检测浏览器版本，而是去检测是否支持你要用的属性。
包括像jQuery这样的热门js库，从1.9版本开始便取消了 [jQuery.browser](http://www.jquery123.com/api/jQuery.browser/) 这样的浏览器版本检测方法，我们可以使用类似 [Modernizr](https://modernizr.com/) 这样的类库。
